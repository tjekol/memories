## Website for my photos.

### Hvordan kjøre?

1. Last ned dependencies 
    ```sh
    pnpm install
    ```

2. Start utviklingsmiljøet
  * Frontend 
      ```sh
      pnpm web:dev
      ```
  * CMS 
      ```sh
      pnpm cms:dev
      ```
